INTELLIIRE
IntelliHire is a frontend interface for an AI-powered hiring platform, focused on clean UX and scalable component design.
 Features
- Role-based authentication UI (Recruiter / Candidate)
- Responsive, centered auth card layout
- Email & password login form
- Social login UI (Google & Facebook)
- Modular React components
- Flexbox-based responsive layout

 Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS (Flexbox, rem-based spacing)
- React Icons
- Flask

 Status
Frontend UI implementation in progress. Backend and authentication logic to be integrated later.

Purpose
Built as a learning-focused, resume-ready project to practice real-world frontend architecture and UI implementation.
